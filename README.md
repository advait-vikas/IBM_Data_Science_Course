▶️ Getting Started
Clone the repository
``` bash
git clone https://github.com/advait-vikas/IBM_Data_Science_Course.git
```
Move into the repository
``` bash
cd IBM_Data_Science_Course
```
Install dependencies
``` bash
pip install -r requirements.txt
```
Launch Jupyter Notebook
``` bash
jupyter notebook
```
